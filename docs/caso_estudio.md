# Caso de estudio DevOps - Escape Room

## Descripción
Este repositorio contiene el videojuego Escape Room desarrollado en Godot, adaptado a un flujo profesional de integración y despliegue continuo.

## Objetivo
Automatizar la validación, pruebas, liberación y generación de artefactos del videojuego mediante GitHub Actions.

## Componentes incluidos
- Workflows de CI/CD
- Scripts de automatización en `devops/`
- Pruebas básicas en `tests/`
- Estructura del proyecto Godot
    
## Flujo implementado
1. Push o Pull Request
2. Validación de estructura
3. Pruebas básicas
4. Exportación de build
5. Empaquetado
6. Generación de artefacto descargable

## Entorno de liberación
- GitHub
- GitHub Actions
- Ubuntu runner
- Bash scripts
- Godot project structure

## Evidencia
El pipeline genera artefactos listos para descarga y validación académica.
